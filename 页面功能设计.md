# **全局开发规范说明**

## **技术栈**

|     |     |
| --- | --- |
| **项目** | **技术/版本** |
| 目标平台 | HarmonyOS 6.0.2 (API 22) |

## **设计规范**

- 采用 HarmonyOS 原生设计风格，不强制模仿 iOS Cupertino 风格，但保持相同的信息层次和交互逻辑
- 底部导航使用 Tabs 组件（barPosition: BarPosition.End），自定义 TabBar 样式
- 页面跳转使用 Navigation 组件 + NavPathStack 管理
- 全局状态使用 AppStorage 存储，页面间通信使用 @Provide/@Consume 或 emitter
- 支持深色模式，使用系统资源引用 $r('sys.color.\*') 和自定义资源
- 列表使用 LazyForEach + IDataSource 实现惰性加载
- 下拉刷新使用 Refresh 组件（API 18+）

# **Prompt: 主页面 (HomePage)**

源文件：lib/page/home_page.dart

目标平台：HarmonyOS 6.0.2 (API 22) / ArkTS / ArkUI

开发工具：DevEco Studio 6.0.2

## **功能描述**

应用的主页面，作为整个应用的导航容器。包含 5 个底部 Tab 页签，分别对应“接下来”“日程”“任务”“学业”“设置”。支持左右滑动切换 Tab，启动时检查应用更新。

## **UI 结构**

- 根组件：Tabs（barPosition: BarPosition.End）
- 5 个 TabContent，分别包含 FlowPage、CalendarPage、TaskPage、ScholarPage、OptionPage
- 底部 TabBar 包含 5 个自定义 Tab 项：图标 + 文定文字
- Tab 图标：时钟、日历、对勾、学校、设置
- 标签栏背景半透明，选中态高亮显示
- 所有子页面同时创建并缓存，通过可见性控制显示哪个

## **交互行为**

- 点击 Tab 图标/文字：切换对应页面
- 左右滑动手势：快速切换 Tab（速度阈值 200px/s 或距离阈值屏幕宽度 15%）
- 启动后延迟 1 秒检查应用更新，有新版本时弹出更新提示对话框
- 更新对话框包含“更新可用”标题、“忽略”和“访问网站”两个按钮

## **数据绑定**

- 当前 Tab 索引：@State currentIndex: number
- 更新检查状态：从 AppStorage 读取 fuse 对象

## **导航关系**

|     |     |     |
| --- | --- | --- |
| **Tab 索引** | **目标页面** | **说明** |
| 0   | FlowPage | 接下来 - 时间流 |
| 1   | CalendarPage | 日程 - 日历视图 |
| 2   | TaskPage | 任务 - DDL管理 |
| 3   | ScholarPage | 学业 - 成绩课表 |
| 4   | OptionPage | 设置  |

## **特殊要求**

- 滑动切换时不应与子页面的垂直滚动冲突，需合理处理手势优先级
- 标签栏需要安全区域适配（避免被系统导航栏遮挡）
- 建议使用 TabsController 控制页签切换，配合 onSelected 事件同步索引

# **Prompt: 接下来页面 (FlowPage)**

源文件：lib/page/flow/flow_view.dart + flow_controller.dart

目标平台：HarmonyOS 6.0.2 (API 22) / ArkTS / ArkUI

开发工具：DevEco Studio 6.0.2

## **功能描述**

展示用户即将到来的事务和自动安排的任务时间块。第一张卡片显示当前/最近的事务详情（含实时进度条），后续卡片列表展示接下来的事务。事务包括课程、考试、用户日程、自动安排的 DDL 时间块。

## **UI 结构**

- 页面标题：“接下来”
- 第一张大卡片：圆角矩形卡片，包含：
    - 事务名称（16px 加粗） + 颜色标记（12x12，根据类型不同颜色和形状）
    - 地点图标 + 地点文字
    - 时间图标 + 时间文字（基于今天的友好时间描述）
    - 倒计时/正计时文字：“离结束还有 XX:XX:XX” 或“开始还有 XX:XX:XX”（等宽数字）
    - 进度条：8px 高度圆角进度条，实时更新（每秒刷新）
- 后续小卡片列表：简化版，显示名称、时间、地点

## **事务类型颜色规则**

|     |     |     |
| --- | --- | --- |
| **类型** | **PeriodType** | **颜色/形状** |
| 自动安排 | flow/user | 根据 UID 生成颜色，圆形 |
| 考试  | test | 粉红色，矩形 |
| 课程  | classes | 根据上课时间着色，圆形 |
| 用户日程 | user | 根据 UID 生成颜色，圆形 |

## **交互行为**

- 点击课程类型卡片：跳转到课程详情页面
- 进度条：实时响应当前时间变化，宽度根据当前时间与开始/结束时间的比例计算

## **数据绑定**

- flowList: Period\[\] —— 从 AppStorage 获取，包含所有当前事务
- timeNow: Date —— 每秒更新，用于计算进度和倒计时
- Period 模型字段：summary、location、startTime、endTime、type、fromUid

## **特殊要求**

- 使用定时器每秒更新时间和进度，注意页面不可见时暂停定时器以节省资源
- 进度条动画需要平滑，使用 animateTo 实现
- 等宽数字显示（tabularFigures），避免数字变化时布局跳动

# **Prompt: 日程页面 (CalendarPage)**

源文件：lib/page/calendar/calendar_view.dart + schedule_view.dart + calendar_controller.dart

目标平台：HarmonyOS 6.0.2 (API 22) / ArkTS / ArkUI

开发工具：DevEco Studio 6.0.2

## **功能描述**

日历视图 + 日视图的双模式页面。日历视图展示月历（支持月/周切换），点击日期后下方显示当天的事件列表；课表视图展示周课表网格。支持添加日程、下拉刷新、今天快速定位。

## **UI 结构**

- 标题栏：左侧显示年月或学期名，右侧操作按钮：
    - “+”添加日程按钮（仅日历模式显示）
    - “今天”快速定位按钮（蓝色文字，仅日历模式显示）
    - 视图切换按钮（日历/课表图标切换）
- 日历控件：中文 locale，周一起始，支持月/周格式切换
- 日期描述行：显示如“春一周”，特殊日期显示绿色边框圆角标签
- 事件列表：每个事件一张圆角卡片，包含颜色标记、名称、时间、地点
- 课表视图：7 列网格（周一至周日）+ 时间列，13 行（08:00-20:30）
- 课程卡片使用 Stack + 绝对定位，自动合并相邻节次的同一课程

## **交互行为**

- 点击日期：更新选中日期，下方刷新事件列表
- 日历格式切换：月/周之间切换
- 添加日程：弹出任务编辑页面（创建固定类型任务）
- 今天按钮：将视图重置为当天
- 视图切换：日历和课表之间切换
- 点击课程/考试卡片：跳转到课程详情页
- 点击固定日程卡片：弹出详情对话框
- 下拉刷新：重新加载数据

## **数据绑定**

- selectedDay: Date —— 当前选中日期
- focusedDay: Date —— 日历聚焦日期
- calendarFormat: 'month' | 'week' —— 日历格式
- events: Map&lt;string, Period\[\]&gt; —— 事件映射表（日期→事件列表）
- viewMode: 'calendar' | 'schedule' —— 视图模式
- specialDates: Map&lt;string, string&gt; —— 特殊日期（节假日、调休）
- Semester.firstHalfTimetable / secondHalfTimetable —— 课表数据（二维数组）

## **特殊要求**

- 日历控件需自定义实现或使用第三方库，支持中文、周一起始、月/周切换
- 选中日期显示半透明蓝色圆形，今天显示半透明灰色圆形
- 日期上有事件时显示小圆点标记
- 课表网格需精确定位课程卡片，支持前半/后半学期切换
- 课程合并算法：相邻节次的同一课程合并为一个卡片

# **Prompt: 任务页面 (TaskPage)**

源文件：lib/page/task/task_view.dart + task_edit_page.dart + task_controller.dart

目标平台：HarmonyOS 6.0.2 (API 22) / ArkTS / ArkUI

开发工具：DevEco Studio 6.0.2

## **功能描述**

DDL 和日程管理页面。分三个分组显示：待办 DDL、已完成 DDL、固定日程。支持左右滑动操作（完成/删除）、点击编辑、新建任务。每次任务变更后自动触发 Flow 重新规划。

## **UI 结构**

- 三个分组标题：“待办”“已完成”“日程”
- 任务卡片：可滑动的圆角卡片，包含：
    - 颜色标记（12x12）
    - 任务名称（18px 加粗）
    - 状态标签（待办/进行中/已结束）
    - 时间信息（截止时间/开始时间）
    - DDL 类型显示进度条（渐变效果）+ “XX% 已完成”文字
- 右下角浮动添加按钮（“+”图标）
- 左滑背景：绿色（完成）/橙色（恢复），含图标
- 右滑背景：红色（删除），含图标

## **任务编辑页面 (TaskEditPage)**

- 导航栏：左侧关闭(X)、中间“编辑任务”、右侧保存(✓)
- 任务类型选择器：“截止日程”/“固定日程”滑动分段控件
- 任务名输入框：placeholder “任务名”
- 开始时间：仅固定日程显示，点击弹出日期时间选择器
- 结束时间：点击弹出日期时间选择器
- 预期用时：仅截止日程，双滚轮选择器（小时 0-999 + 分钟 0-59）
- 地点输入框：placeholder “地点”
- 说明输入框：placeholder “说明”
- 重复类型：仅固定日程，“不重复/每天/每周/每月/每年”
- 重复截止日期：仅固定日程+有重复时显示
- 阻挡安排开关：仅固定日程
- 删除按钮：红色“删除任务”

## **交互行为**

- 右滑：删除任务（真正移除）
- 左滑：标记完成/恢复未完成（不移除，只更新状态）
- 点击卡片：跳转到任务编辑页面
- 添加按钮：弹出任务编辑页面（创建新任务）
- 保存验证：检查时间合法性（开始&lt;结束、重复截止&gt;开始、持续时间不过长）

## **数据绑定**

- taskList: Task\[\] —— 全部任务列表
- todoDeadlineList / doneDeadlineList / fixedDeadlineList —— 分组过滤后的列表
- Task 模型字段：summary、type、status、startTime、endTime、timeNeeded、timeSpent、progress、location、description、repeatType、repeatPeriod、repeatEndsTime、blockArrangements

## **特殊要求**

- 滑动操作需平滑动画，阈值 0.25，动画时间 300ms
- 每秒定时刷新任务状态（清理已删除、刷新状态、处理重复日程过期）
- 任务变更后自动触发 Flow 重新规划
- 编辑页面作为 Modal 弹出，保存后返回编辑后的 Task 对象

# **Prompt: 学业页面 (ScholarPage)**

源文件：lib/page/scholar/scholar_view.dart + scholar_controller.dart

目标平台：HarmonyOS 6.0.2 (API 22) / ArkTS / ArkUI

开发工具：DevEco Studio 6.0.2

## **功能描述**

学业信息主页，展示 GPA、成绩概览、课程统计、作业列表、实践学分等。本科生显示 4 个模块，研究生显示 2 个模块。支持学期切换、下拉刷新、搜索跳转。

## **UI 结构**

- 固定头部：标题“学业”(24px) + 搜索框 + 学期选择器（水平滚动）
- 搜索框：placeholder “搜索课程、事项...”，始终禁用焦点（点击跳转搜索页）
- 模块一：成绩模块（绿色额头条）
    - 6 个双行卡片：五分制GPA、获得学分、四分制GPA、主修均绩、主修学分、百分制
    - 更新状态：勾/叹号 + “更新于 X 分钟前”
- 模块二：课程模块（青色额头条）
    - 统计：课程数、学分、考试数（可点击）
    - 2 个课时卡片：前半/后半学期课时（可点击）
- 模块三：作业模块（品红色额头条，仅本科生）
    - 统计：总计、一天内、本周截止
    - 作业卡片水平列表：课程名、作业名、截止时间
- 模块四：实践模块（桃色额头条，仅本科生）
    - 三个分数：二课分、三课分、四课分
- 空状态：“下拉刷新”或“未登录”
- 错误处理：错误提示 + “重新获取数据”按钮

## **交互行为**

- 搜索框点击：跳转到搜索页面
- 学期选择：点击切换当前学期，带动画效果
- 成绩卡片点击：跳转到成绩详情页（全屏对话框）
- 课程数点击：跳转到课程列表页
- 考试数点击：跳转到考试列表页
- 课时卡片点击：跳转到课表页（前半/后半）
- 下拉刷新：调用 fetchData() 从网络刷新数据

## **数据绑定**

- semesters: Semester\[\] —— 所有学期列表
- semesterIndex: number —— 当前选中学期索引
- gpa: number\[\] —— GPA 列表（五分制、4.3、原始四分制、百分制）
- todos: Todo\[\] —— 作业列表（按截止时间排序）
- hideHomeGpa: boolean —— 是否隐藏绩点（显示为 \*\*\*）
- isGrs: boolean —— 是否为研究生（决定显示哪些模块）

## **特殊要求**

- 学业页→成绩详情页需要 Hero 共享元素动画（tag: 'gradeBrief'）
- 学期选择器带动画效果，选中态高亮
- 每秒更新“更新于 X 分钟前”显示
- 绩点隐藏时将数字替换为 \*\*\*
- 使用 CustomScrollView + Slivers 实现固定头部 + 下拉刷新

# **Prompt: 设置页面 (OptionPage)**

## **可用工作时段编辑页 (AllowTimeEditPage)**

- 标题“编辑可用工作时段”
- 时段列表：每个显示“HH:mm - HH:mm”，含编辑和删除按钮
- 添加按钮：“添加一个时段”，默认 08:00-12:00

# **Prompt: 搜索页面 (SearchPage)**

源文件：lib/page/search/search_view.dart + search_controller.dart

目标平台：HarmonyOS 6.0.2 (API 22) / ArkTS / ArkUI

开发工具：DevEco Studio 6.0.2

## **功能描述**

全局搜索页面，支持模糊搜索所有学期的课程。输入关键词实时过滤课程列表。

## **UI 结构**

- 固定搜索栏：CupertinoSearchTextField，placeholder “搜索课程、事项...”，自动聚焦
- 搜索结果列表：CourseBriefCard 课程简要卡片

## **交互行为**

- 输入搜索：onChanged 实时更新搜索结果
- 自动聚焦：进入页面时自动弹出键盘
- 点击课程卡片：跳转到课程详情页

## **数据绑定**

- searchWord: string —— 搜索关键词
- allCourses: Course\[\] —— 从所有学期汇总的课程列表
- courseResult: Course\[\] —— 模糊匹配结果（name.contains()）

## **特殊要求**

- 空搜索词返回空列表
- 监听 scholar 数据变化自动刷新课程列表
